# 浏览器相关

## hover 效果如何调试

```js
// 控制台，执行 debugger，然后触发 hover 即可
setTimeout(() => {debugger}, 3000)

// 或者是在控制台 元素审查-选中 `:hov`
```

## 不用 mock 接口修改接口返回

1. `f12` 网络下，在对应的接口右键，点击替换内容
2. 然后选择存储数据的文件夹
3. 以 `Json` 格式编辑返回的数据

**注意：**

1. 只能修改接口返回，对应触发服务器的接口处理还是会进行，不是拦截接口，只是模拟数据返回
2. 修改的接口文件，可以在源代码 tab 下修改和禁用
