# 项目定制化 rules

以 `cursor` 为例，在项目创建文件 `.cursor\rules\${自定义名称}.mdc`

示例文件

````
---
<!-- 项目描述 -->
description: xxx
<!-- 约束生效文件 -->
globs: *.vue 
<!-- 是否总是生效 -->
alwaysApply: false

---

# 项目编码约定

## Vue文件模版

新创建vue文件使用这个模版

```vue
<template>
  <view></view>
</template>
<script setup lang="ts"></script>
<style scoped lang="less">
*,
view,
text,
input,
textarea,
button,
image {
  box-sizing: border-box;
}
</style>
```

### Props定义

```javascript
// ✅ 详细props定义
props: {
  userId: {
    type: String,
    required: true
  }
}

// ❌ 避免简单数组形式
props: ['userId']
```

### 部分方法示例

#### 小程序分享功能

在setup语法下，需要创建额外的script脚本

```vue
<script lang="ts">
export default {
  onShareAppMessage() {
    return {
      title: '分享',
      path: '/pages/index/index',
    };
  },
};
</script>
```

选项式下不用单独的script脚本

### 小程序样式穿透

在setup语法下，需要创建额外的script脚本

```vue
<script lang="ts">
export default {
options: {
    styleIsolation: 'shared',
  },
};
</script>
```

选项式下不用单独的script脚本

  // 实现逻辑

```
````
